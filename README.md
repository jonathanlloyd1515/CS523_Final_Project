# CS523 Final Project

* Jupyter Notebooks (NN_GN_Experiment_1.ipynb and NN_GN_Experiment_2.ipynb) contain relevant code 
* Other files are helper scripts for feature extraction and preprocessing

## NN_GN_Experiment_1.ipynb
* Contains method for replicating "Genetic CNN" by Xie et al. (2017).
* CNN are dynamically constructed by decoding bit strings
* Bit strings evolved with PyGad genetic algorithm

## NN_GN_Experiment_2.ipynb
* Contains method for macroscale optimization of CNN
* CNN are dynamically constructed by decoding bit strings
* Bit strings evolved with PyGad genetic algorithm
